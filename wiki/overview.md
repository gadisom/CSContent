## 앱 현황

CSContent 앱은 다음 핵심 기능을 갖추고 있다:
- **퀴즈** — 퀴즈 생성 및 풀기 기능 구현 완료
- **개념 설명** — 개념별 설명 콘텐츠 기능 구현 완료

이 위키의 목표는 **어떤 콘텐츠를 만들지** 기록하고, **개념 간 연결**을 유지하며, **제작 공백**을 시각화하는 것이다.

## 전략적 방향

### Phase 1 — 기초 탄탄히
자료구조 + 알고리즘의 핵심 개념 20개를 먼저 커버한다.
퀴즈와 설명이 세트로 존재해야 완성.

### Phase 2 — 시스템 이해
운영체제 + 네트워크 + 데이터베이스로 확장.
면접 대비 비중이 높은 개념 중심.

### Phase 3 — 심화
시스템 설계, 비교 콘텐츠, 난이도별 경로 구성.
