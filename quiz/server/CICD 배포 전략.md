#### 빈칸 | [657]
블루그린 배포에서 롤백 시 수정이 필요한 것은 ___ 뿐이다.
> 로드밸런서
> Blue(구버전)와 Green(신버전) 환경이 모두 준비된 상태에서 로드밸런서 설정만 되돌리면 즉시 롤백이 완료된다.

#### 빈칸 | [658]
Nginx에서 동일 클라이언트를 항상 같은 서버로 연결해 세션 유지에 유리한 로드밸런싱 방식은 ___ 이다.
> IP Hash
> IP Hash는 클라이언트 IP를 해시해 항상 동일한 서버로 라우팅한다. 세션 기반 인증처럼 동일 서버 연결이 필요한 경우에 적합하다.
