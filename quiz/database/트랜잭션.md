#### 객관식 | [401]
ACID 속성 중 'I'가 의미하는 것은?
1. Integrity
2. Index
3. ✅ Isolation
4. Integration
> Isolation(격리성): 동시에 실행되는 트랜잭션이 서로 영향을 주지 않아야 합니다.

#### 객관식 | [403]
트랜잭션 격리 수준 중 가장 엄격한 것은?
1. READ UNCOMMITTED
2. READ COMMITTED
3. REPEATABLE READ
4. ✅ SERIALIZABLE
> SERIALIZABLE은 완전한 순차 실행을 보장하며 동시성은 가장 낮습니다.

#### 객관식 | [699]
ACID 속성 중 트랜잭션이 완료되면 결과가 영구히 저장되고, 중간에 오류가 나도 작업 전체가 취소되는 특성을 모두 고르시오. (2개)
1. ✅ 지속성(Durability)
2. 일관성(Consistency)
3. ✅ 원자성(Atomicity)
4. 격리성(Isolation)
> 원자성(Atomicity)은 트랜잭션 전체가 성공하거나 전체 실패함을 보장하고, 지속성(Durability)은 커밋된 결과가 장애 후에도 유지됨을 보장합니다.
