#### 객관식 | [401]
ACID 속성 중 'I'가 의미하는 것은?
1. Integrity
2. Index
3. ✅ Isolation
4. Integration
> Isolation(격리성): 동시에 실행되는 트랜잭션이 서로 영향을 주지 않아야 합니다.

#### 객관식 | [403]
트랜잭션 격리 수준 중 가장 엄격한 것은?
1. READ UNCOMMITTED
2. READ COMMITTED
3. REPEATABLE READ
4. ✅ SERIALIZABLE
> SERIALIZABLE은 완전한 순차 실행을 보장하며 동시성은 가장 낮습니다.
