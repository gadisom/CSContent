#### 빈칸 | [508]
DNS의 역할은 도메인 이름을 컴퓨터가 찾을 수 있는 ___ 로 변환하는 것이다.
> 주소
> DNS(Domain Name System)는 사람이 읽기 쉬운 도메인을 컴퓨터가 사용하는 IP로 변환합니다.

#### 객관식 | [572]
DNS에 대한 설명으로 가장 알맞은 것은?
1. 패킷 암호화를 수행한다
2. 라우팅 경로를 계산한다
3. 포트 번호를 할당한다
4. ✅ 도메인 이름을 IP 주소로 변환한다
> DNS는 인터넷의 전화번호부 역할을 합니다.

#### 빈칸 | [574]
DNS 레코드는 도메인을 컴퓨터가 찾아갈 수 있는 ___ 에 연결한다.
> 주소
> A 레코드는 도메인을 IPv4 주소에 매핑합니다. IPv6는 AAAA 레코드를 사용합니다.
