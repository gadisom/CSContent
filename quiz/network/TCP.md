#### 객관식 | [503]
TCP 3-way Handshake의 올바른 순서는?
1. ACK → SYN → SYN-ACK
2. ✅ SYN → SYN-ACK → ACK
3. SYN-ACK → SYN → ACK
4. SYN → ACK → SYN-ACK
> SYN(연결 요청) → SYN-ACK(응답) → ACK(확인) 순으로 연결이 수립됩니다.
