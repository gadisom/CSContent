#### 객관식 | [306]
가상 메모리(Virtual Memory)의 주된 목적은?
1. CPU 속도 향상
2. ✅ 실제 메모리보다 큰 프로세스 실행
3. 디스크 용량 절약
4. 캐시 일관성 유지
> 가상 메모리는 보조 기억 장치를 활용해 물리 메모리 제한을 넘어선 실행을 가능하게 합니다.

#### 객관식 | [590]
가상 메모리에 대한 설명으로 가장 알맞은 것은?
1. 물리 메모리보다 무조건 빠르다
2. TLB가 필요 없다
3. 페이지 교체 정책이 성능과 무관하다
4. ✅ 페이지 폴트가 발생하면 디스크 I/O로 지연될 수 있다
> 주소 변환과 페이지 교체는 메모리 성능의 핵심 요소입니다.
